CN Google App Engine Sample

This is an example showing how to use Consumer Notebook's Grids API. It's open-source and runs on GAE.

You can download, run, and deploy this yourself: just grab the code from https://github.com/consumernotebook/cn-gae-sample/ and follow the instructions in the README file.

For more info and code samples, see http://developers.consumernotebook.com.
